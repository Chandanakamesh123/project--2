# Factory-to-Customer Shipping Route Efficiency Analysis for Nassau Candy Distributor

## Project Overview
This project focuses on analyzing and improving the efficiency of factory-to-customer shipping routes for Nassau Candy Distributor. The system uses Data Analytics and Machine Learning techniques to identify delayed shipments, analyze transportation performance, and improve logistics efficiency.

---

## Purpose of the Project
The main purpose of this project is to analyze shipping routes and reduce delivery delays by improving route efficiency and logistics management.

---

## Objectives
- Analyze shipment and delivery performance
- Identify delayed shipping routes
- Compare shipment modes and vendor performance
- Predict delivery delays using Machine Learning
- Improve transportation efficiency

---
